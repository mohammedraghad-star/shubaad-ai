# SHUBAAD System Architecture v1

SHUBAAD integrates FTIR spectroscopy, GIS layers, soil metadata, and machine-learning models for salinity, soil organic matter, drought, and environmental resilience assessment.

## v1 Modules

1. Spectral Engine
2. GIS Engine
3. AI Decision Engine
4. Scientific Reporting Engine

## Deployment

- Frontend: Vercel, root directory `frontend`
- Backend: Render/Railway, root directory `backend`
- Database: Supabase PostgreSQL/PostGIS
